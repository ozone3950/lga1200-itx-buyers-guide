# Z490 ITX Roundup

Here is an all-together, easy to read roundup of the relevant Z490 ITX boards.


## ASRock Z490 Phantom Gaming ITX
- 6+3 Phase VRM
  - Controller: ISL69269
  - VCore: 6x ISL99390 90A
  - SA/IO: 3x ISL9927 60A

- Twin 4+2 Phase VRM
  - Controller: ASP1405

  - SA: 2x TDA21462 60A
